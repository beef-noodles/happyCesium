# 基于WebSocket的实时数据传输

![Cesium Air WebSocket](./img/air.png)

## 获取

```shell
git clone -b websocket https://github.com/guzhongren/FrontEndStudy.git
```

## 安装依赖

```shell
$ cd FrontEndStudy
$ npm install
# yarn
```

## 启动服务端

```shell
$ npm run server
# yarn run server
```

## 启动浏览器

```shell
$ npm run browser
# yarn run browser
```
